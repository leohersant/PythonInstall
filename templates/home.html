How to install Python

Installing Python

https://www.digitalocean.com/community/tutorials/how-to-install-python-3-and-set-up-a-programming-environment-on-an-ubuntu-20-04-server-fr

Check if is installed (on ubuntu)

    python --version
    python3 -V

Install Python Interface Package (package manager)

    sudo apt install -y python3-pip

Other libraries

    sudo apt install -y build-essential libssl-dev libffi-dev python3-dev

Install tools for environments

    sudo apt install -y python3-venv
    sudo pip3 install virtualenv

Test Python

    mkdir Python/Test
    cd Python/Test
    create a setup.sh file
    sudo chmod +x setup.sh
    ./setup.sh
